Ajkay Örkény [orkeny]
A Windows XP a multiuser lehetőségek miatt nem kapcsolja be.
Registry hackeléssel (a rendszerleíró adatbázis buherálásával) meg lehet oldani, de ezt inkább a rendszergazda csinálja (csak saját felelősségre!):
Így kell:
- Start menü / Futtatás: regedit
- HKEY_USERS / .Default / Control Panel / Keyboard.
- az "InitialKeyboardIndicators" értékét átírni 0-ról 2-re.
Újraindítás után a NumLock be lesz kapcsolva.
