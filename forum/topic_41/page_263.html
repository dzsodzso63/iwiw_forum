12669:
A hover az az anchor (azaz a klikelhető link) egyik állapota a 4 közül, és valóban, tulképp a link "onmouseover"-je, de ne keverjük a hovert az onmouseoverrel, mert az előbbi mint írtam, egy link állapot, a másik egy (script) esemény, amit máshoz is hozzá lehet rendelni, ha jül mondom, de majd szóljon bele más kolléga is.
Példa:
Az alábbi példában a "majonéz" osztályú link betűszíne (és csak az) megváltozik, ha ráviszik az egeret:
------ html body részben: ------------
<a href="arlista.html" class="majonez">Árlistánk</a>
------ a css-ben: --------------------
a:link.majonez {
color: #BBC0CC;
font-size: 8pt;
font-family: Verdana;
font-weight : bold;
text-decoration: none;
}
a:hover.majonez {
color: #AFE9FE;
}
-----------------------------
Nem tudom, ez segít-e.
Ha egy linket dobozba akarsz tenni, és az egész dobozt akarod változtatni, akkor a css-ben az adott anchort blokká kell alakítani, pl.:
a:link.majonez {
display: block;
color: #BBC0CC;
font-size: 8pt;
font-family: Verdana;
font-weight : bold;
text-decoration: none;
}
És akkor lehet definiálni szélességet, magasságot, háttérszínt, háttérképet, és a hover-es részben azokat változtatni, stb.
