Kedves Dorka,
Ha Wordben dolgozol, akkor az Eszközök =Tools) menü Beállítások (Options) parancsával kapsz egy párbeszédablakokt. Azon keresd meg a Mentés (Save) szegélycédulával jelzett részt, és ott valószínűleg találsz egy számlálót, amivel megadhatod az automatikus mentések intervallumát.
Megfontolandó, hogy lehetőleg ne tarts nyitva egyszerre több dokumentumot, csak akkor, ha ténylegesen szükséged van rájuk (pl. referenciaként vagy egyikből a másikba történő adatátvitel miatt). A már megnyitott dokumentumok ismételt megnyitása a Wordöt "zavarba hozza", ezért kérdez rá, hogy mi legyen a már megnyitott példány változásaival.
Azt is javaslom, hogy NE bízz az automatikus mentésben!
Az automatikus mentés NEM igazi mentés.
A Word ilyenkor csak egy ideiglenes fájlban rögzíti a dokumentum pillanatnyi állapotát, és csak akkor írja ezt ki "igazán" a dokumentumfájlba, ha a Wordöt bezárod (az ideiglenes fájlt pedig törli), tehát csak ekkor kerülnek igazán biztonságba az adatok.
Jól teszed, ha egy-egy nagyobb szövegrész bevitele vagy formázási, szerkesztési műveletek után nyomsz egy Ctrl-S billentyűkombinációt, vagy a Fájl menüből a Mentés parancsra (illetve a Szokásos eszköztár 3. ikonjára, a floppyra) kattintasz. Ezek által ténylegesen elvégzed a mentést, így a részmunkafázisok is tárolásra kerülnek, és neked is megnyugtatóbb.
Jó szerkesztést!

The new method Font.createFont(int, InputStream) provides the ability to add fonts to the JavaTM 2 Virtual Machine* (JVM) at runtime. This font is not persistent upon termination of the JVM and is only available to the creator of the Font. At this time, only TrueTypeTM fonts can be created at runtime.
The following code sample illustrates how to dynamically load the TrueType font Arial from a file:
\tFile file = new File("Arial.ttf");
\tFileInputStream fis = new FileInputStream(file);
\tFont font = Font.createFont(Font.TRUETYPE_FONT, fis);
Similarly, to load the font from a URL:
\tURL url = new URL("Arial.ttf");
\tInputStream is = url.openStream();
\tFont font = Font.createFont(Font.TRUETYPE_FONT, is);
(http://java.sun.com/j2se/1.3/docs/guide/2d/new_features.html)
