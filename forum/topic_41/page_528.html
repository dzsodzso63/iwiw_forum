Ajkay Örkény [orkeny]
Kooka:
Akkor nézzük sorban:
A servicéket (rendszer szolgáltatások) most hagyjuk, azokról írtam korábban, de az ott szereplő Messenger szolgáltatásnak valóban semmi köze a Windows vagy az MSN Messengerekhez.
A Windows Messenger az XP része, és eredetileg arra szolgál, hogy helyi (vállalati) hálózati környezetben használják egymás közt. keveset tud, elavultnak számít, és helyette inkább mindenki a MSN Messenger nevű, ingyenesen letölthető programot használja.
A MSN Messenger automatikus elindulásának letiltása:
1. Nyisd ki a Messenger főablakát (ami a kontatkok listáját mutatja)
2. kattints a Tools (Eszközök) menüre
3. kattints azon belül az Options (Beállítások) parancsra
4. a párbeszédablakban kattints a General (Általános) fülre
5. a legelső jelölőnégyzetből (Messenger automatikus futtatása) szedd ki a pipát
6. kattints az OK gombra
Ezek után a MSN Messenger már nem fog elindulni magától.
A WINDOWS Messenger viszont még elindulhat, ennek letiltása:
1. Nyisd ki a Windows Messenger főablakát (ami a kontatkok listáját mutatja)
2. kattints a Tools (Eszközök) menüre
3. kattints azon belül az Options (Beállítások) parancsra
4. a párbeszédablakban kattints a Preferences (balról a 3.) fülre
5. a legelső jelölőnégyzetből (E program futtatása a Windows indulásakor) szedd ki a pipát
6. kattints az OK gombra
Ezzel elvileg ki van iktatva a Windows Messenger automatikus elindulása is.
Ha a fentiek ellenére mégis elindul valamelyik, akkor szólj. Akkor az MSCONFIG nevű beállítóprogrammal lehet megoldani, de addig abba most nem akarok belemenni.
