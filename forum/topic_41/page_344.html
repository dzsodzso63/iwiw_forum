10229: a css osztalyokbol csinalsz mas mas szinut es az oldalon dinamikusan adod meg, hogy melyik css osztalyt hasznalja. pl:
css:
.kek {
bgcolor = #0000ff;
}
.piros {
bgcolor = #ff0000;
}
html:
<body class="<%=user.szin%>">
Itt most JSP szintaktikat hasznaltam, de PHP, vagy Perlben is hasonlo
A user.szin termeszetesen "kek" vagy "piros" lehet, igy az oldal forditasakor a class="kek" vagy class="piros" lezs a HTMLben, igy a user a sajat szinet fogja latni.
Azt, hogy a user milyen szint szeret el lehet menteni pl Cookieban, vagy szebben megcsinalva adatbazisban :-)
Ha a szineket csak promt akarod megvaltoztatni, azaz nem akarod eltarolni a beallitasokat, akkor egy kis DHTML koddal(+Javascript) ki lehet dinamikusan cserelni a css osztalyokat, meg barmi mast is.
