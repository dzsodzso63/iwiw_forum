Hm érdekes, pedig a NumLock állapotát még az én múzeális értékű gépemen (pontosabban alaplapomon) is lehet állítani, talán csak el van dugva valahol.
Van még egy tippem, Éva, de ez
--------------------------
nem kezdőknek való, és csak SAJÁT FELELŐSSÉGRE!
--------------------------
mindenképpen csak szakemberrel csináltasd meg, aki már látta a RegEditet belülről (itt nem szabad hibázni, mert a Windows lelkét szerelgeted ezzel).
Tehát:
- Start menü
- Futtatás: beírni: regedit (Enter)
A megjelenő Rendszerleíróadatbázis-szerkesztőben:
- bal oldali hierarchiában elmenni ide: HKEY_CURRENT_USERS/Control Panel/Keyboard (rákattintatni a Keyboard mappára)
Az ekkor a jobb oldali mezüben megjelenő kulcsok közül a
InitialKeyboardIndicators
nevűre duplát kattintani,
A kinyíló Edit String ablakban, az alsó sorban levő
Value (Érték)
mezőbe beírni: 2
(semmi mást, se szóköz, se pont se semmi)
Beírás után:
- OK-ra kattintani
- bezárni a Registry Editort
- újraindítani a Windowst
- és reménykedni :)
